马上开始使用    或者， 离线下载
阅读模式
逐一适配 120+ 个网址；
自动提取 标题，描述，正文，媒体 （ 图片 / 视频 ） 等资源；
自动生成符合中文阅读习惯的页面；
零干扰，沉浸式阅读，如杂志般的阅读体验；
适合深入阅读；
聚焦模式
不改变当前页面的结构，仅仅高亮需要阅读的部分；
适合临时阅读 或 阅读模式未适配的网站；
多种主题模式
白练 白磁 卯之花色 丁子色 娟鼠 月白 百合 紺鼠 黒鸢
高度定制化，可编程

HTML 节点 / jQuery 语句 / 正则表达式 / 任意段落
