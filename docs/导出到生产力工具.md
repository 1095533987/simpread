> 导出到生产力工具 从 1.0.3 版本就已经存在 且是 **简悦独有** 的功能。

支持服务
---
>  `语雀` `坚果云` `Google 云端硬盘`  `Dropbox` `印象笔记/Evernote` `Onenote` `Linnk` `Pocket` `Instapaper` `WebDAV`

导出类型
---

#### 保存链接

> `Linnk` `Pocket` `Instapaper`

#### 保存为 Markdown

> `语雀` `坚果云` `Google 云端硬盘`  `Dropbox` `WebDAV`

#### 保存为 HTML

> `Onenote`  `印象笔记/Evernote`
