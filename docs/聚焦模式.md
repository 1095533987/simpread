含义
---
> 不改变当前页面的结构，仅仅高亮需要阅读的部分，适合 `临时阅读` 或者 `未适配阅读模式` 的网站；

如何使用
---

`聚焦模式` 会自动获取当前鼠标所在的段落并高亮，适合任意页面。

- 在需要高亮的区域，右键选择 `简悦 - SimpRead` → `聚焦模式`；

- 快捷键；（默认为 `A S` )
