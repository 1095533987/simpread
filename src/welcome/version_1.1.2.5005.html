恭喜，您的简悦已升级到最新 1.1.2.5005 版本

简悦 的每一次 「进化」 都意味着，为您提供更优质的阅读体验。

本次升级为您准备了以下的新功能：

  • 全新的词法分析引擎 2.0 beta
  • 更加方便的右键菜单
  • 分享卡
  • 全部功能

全新的词法分析引擎 2.0 beta
智能识别依赖与 词法分析引擎，在本次更新中升级为 2.0 版本，详细说明请访问 词法分析引擎
识别范围：

1Wordpress · Hexo · Ghost · Discuz 等博客 / 论坛的页面

2LaTeX 解析 · TXT 格式 · Markdown 格式 · 代码段

3智能修复某些图片被错误居中的情况

4智能识别图片的相对地址

由于当前处于测试版，在某些极端条件下疑似会出现问题，因此可将其手动关闭。

1选项页 → 高级设定 → 词法分析引擎 关闭选项即可

更加方便的右键菜单
右键菜单增加了：当前页面加入到稍后读 · 当前页面加入到白名单 · 当前页面加入到黑名单 · 当前页面加入到排除列表。
注意：默认这些功能为关闭状态；启用方法：选项页 → 高级设定 → 右键菜单，分别开启即可。

分享卡
下图就是分享卡的功能，通过此功能，方便分享到任意社交网络。
使用方法：

1控制栏 → 动作 → 共享 → 分享卡 icon

2在阅读模式上按住鼠标左键不放，进行高亮操作

3会自动弹出如下界面，选择下载即可。
