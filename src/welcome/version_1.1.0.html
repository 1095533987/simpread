恭喜，您的简悦已升级到最新 1.1.0 版本

简悦 的每一次 「进化」 都意味着，为您提供更优质的阅读体验。

本次升级为您准备了以下的新功能：

  • 站点编辑器
  • 更灵活的适配站点样式
  • 站点适配源
  • 站点管理器
  • 全部功能

站点编辑器
从 1.1.0 开始，你可以使用 站点编辑器 对任意适配站点进行 修改，删除，甚至 新增 适配站点了。

更灵活的适配站点样式
从现在开始，支持不同的适配站点定制不同的样式，你可以认为这是简悦版的 Stylish

站点适配源
从现在开始，简悦增加了全新的 站点适配源 系统，可以 导入 / 管理 第三方站点源了。

1选项页 → 高级设定 → 站点适配器

站点管理器
自定义的站点太多，不好管理？1.1.0 还提供了更加强大的 站点管理器，方便管理全部的站点。

1选项页 → 高级设定 → 站点适配器 → 适配管理器
